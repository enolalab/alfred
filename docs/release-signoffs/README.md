# Release Sign-offs

Place real release sign-off files in this directory.

Guidelines:

- one sign-off file per release decision
- use a descriptive filename, for example:
  - `2026-04-03-prod-canary-1.md`
- do not use the template file itself for a real release decision
- do not use `README.md` or a placeholder path such as `replace-with-real-signoff.md`
- reviewer names and the final decision must be filled in before running the release-readiness workflow
